## 📌 Overview
MediCore HMS is an AI-powered hospital management system designed to improve healthcare operations through intelligent automation, forecasting, and computer vision.

This repository contains the **official landing page website** for the MediCore HMS research project.

---

## 🚀 Features

- 🔐 Role-Based Access Control (Admin & Doctor portals)
- 📊 AI Drug Consumption Forecasting (CatBoost)
- ⏳ Drug Expiry Risk Prediction
- 👨‍⚕️ Patient Volume Forecasting
- 🏥 Equipment Demand Forecasting
- 📷 Image-Based Drug Identification (OCR + CNN)
