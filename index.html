Platform Features

Everything a modern hospital needs

Eight intelligent modules built to work together — from real-time inventory to AI-powered demand forecasting and patient safety.

🔒

Role-Based Access Control

Separate Admin and Doctor portals with JWT-based authentication. Every endpoint enforces role claims so sensitive data never leaks across roles.

Admin + Doctor Portals
📈

AI Drug Consumption Forecast

CatBoost gradient-boosting models predict monthly drug consumption per category, enabling proactive procurement and preventing stockouts before they happen.

7.3% MAPE accuracy
⏳

Drug Expiry Prediction

Every batch is automatically risk-classified as High, Medium, or Low risk based on days remaining. Pharmacists get clear action signals, not raw numbers.

3-tier Risk Classification
👥

Patient Volume Forecasting

Predict daily outpatient arrival counts using temporal features — weekday, month, seasonal index, and public holiday flags — with 8.1% MAPE accuracy.

8.1% MAPE accuracy
🛠

Equipment Demand Forecasting

Predict equipment utilisation peaks before they cause bottlenecks. Dynamic resource allocation informed by historical usage and upcoming patient volume projections.

Proactive Procurement
📷

Image-Based Drug Identification

Photograph a medicine package and the system extracts the drug name and expiry date automatically using MobileNetV2 CNN + Tesseract OCR. No typing required.

F1-Score 0.935
⚠

Drug-Drug Interaction

Automatically detects potentially dangerous combinations when multiple drugs are prescribed simultaneously. The system cross-references a comprehensive interaction database, flags contraindicated pairs in real time, and alerts the prescribing physician before the order is confirmed — preventing adverse drug events at the point of care.

Real-Time Interaction Alerts
🧠

AI-Powered Patient Risk Profiling & Prescription Safety

Builds a dynamic risk profile for every patient by analysing medical history, active diagnoses, allergy records, and current medications. Machine learning models score each new prescription against the patient's profile, surfacing high-risk prescriptions, dosage anomalies, and allergy conflicts before they reach the pharmacy — turning reactive error-catching into proactive patient safety.

Proactive Prescription Safety
How It Works

Up and running in four steps

From deployment to your first AI-powered forecast in under a day.

1

Deploy on AWS

One-command Docker deployment to your AWS EC2 instance. PostgreSQL provisions automatically on RDS with backups enabled.

2

Import Your Data

Migrate existing drug inventory and patient records via our CSV import tool or REST API. Historical data trains your custom forecast models.

3

Configure Roles

Create Administrator and Doctor accounts. Set inventory thresholds, forecasting horizons, and alert preferences from the admin panel.

4

Start Forecasting

AI models activate immediately. Receive drug expiry alerts, consumption forecasts, and patient volume predictions on your dashboard daily.

AI Vision Module

Photograph it. Done.

The image processing module eliminates manual data entry during stock receiving — the single biggest source of pharmacy transcription errors.

From a photo to structured data in 2.3 seconds

A pharmacist photographs the incoming medicine batch. MediCore HMS reads the drug name and expiry date automatically, cross-checks it against the Sri Lanka National Formulary (3,400 entries), and pre-fills the inventory form. The pharmacist reviews and confirms — no keyboard needed.

✓
Supports English, Sinhala, and Tamil pharmaceutical labels
✓
Automatic fallback to full-image OCR for unusual layouts
✓
Formulary fuzzy-match flags unregistered or suspect products
✓
87% reduction in expiry date transcription errors in pilot testing
